**README FILE**

**Name of Web Store:** IIE booksheilf 
**Project Purpose:**
This web application, named "IIE booksheilf " is designed to facilitate the buying and selling of second-hand textbooks exclusively for students within our college. In response to the challenges posed by the COVID-19 pandemic and the shift to online learning, this platform aims to provide a convenient and secure online marketplace for students to trade used textbooks.

**Features:**
- User registration and authentication.
- Librarian verification for sellers (students).
- Buying and selling functionality.
- Shopping cart for users to manage selected items.
- Admin functionalities for book and user management.
- Book selling request feature for users to add new books.

**Installation Instructions:**

**Step 1: Launch the WAMP server:**
Ensure that WAMP (or any other suitable web server) is installed and running on your local machine.


**Step 2: Load Data:**
- Populate the database with tables and sample data using the console or phpMyAdmin.
- Use the provided file named bookstore.sql for data on each base table and sample data.

**Step 4: Launch the Application:**
- Open your web browser and enter the address where the application is hosted.
- The login page will appear, allowing users to register, login, and perform other functionalities.
